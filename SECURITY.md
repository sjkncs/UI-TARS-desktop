| Version | Supported          |
| ------- | ------------------ |

## Reporting a Vulnerability

- **For issues in the original project:** please report to https://github.com/bytedance/UI-TARS-desktop/security.
- **For issues specific to this fork:** please report to https://github.com/sjkncs/UI-TARS-desktop/issues.

We will get in touch with you shortly.

## Security Advisories

We will publish security advisories for the latest version.
